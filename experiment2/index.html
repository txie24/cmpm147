Experiment 2 - Generative Landscape

Description

This generative artwork reimagines a desert landscape inspired by the Mojave dunes. The visual is procedurally generated using layered Perlin noise to create realistic terrain shapes, parallax scrolling, and environmental depth.

The scene consists of multiple visual layers: a parallax-shifting sky with dynamic clouds, two layers of mountains with sharp peaks created using modified noise, and three layered dunes in the foreground. Each dune has a unique shape and color gradient, giving depth and texture. From the peak of each dune, a wiggly vertical line is drawn down, casting a shadow gradient along its left side to simulate lighting and elevation.

Technical

This project was built using p5.js and leverages Perlin noise to procedurally generate natural-looking terrain features. The composition includes:

  • Sky and Clouds: A static sky background with wispy cirrus clouds that loop and shift with parallax.
  • Mountains: Two distinct mountain layers use noise-based elevation and a sharpening function to create jagged peaks. The foreground mountain has a different noise frequency and offset to make it visually distinct.
  • Dunes: Three âˆª-shaped inverted dune layers are created using a custom slope and noise algorithm. Each dune has its own color gradient and randomized ridge offset to avoid repetition.
  • Wiggly Shadows: From the highest point of each dune, a smooth, noise-driven wiggly line is drawn downward. A clipped gradient shadow is rendered to the left of this line, simulating soft shading caused by dune curvature.
  • Parallax: Different layers shift at various speeds in response to mouse movement to give a sense of depth and perspective.

The landscape responds interactively to user input through a "Reimagine" button that regenerates the scene using a new seed, introducing variation in every refresh.
